# Claw Design — Landing Page Skill

生成 Landing Page 落地页（响应式 HTML）。

## 使用方式

```bash
npx tsx scripts/generate.ts --content "产品或活动描述"
```

参数：
- `--content`：落地页内容的自然语言描述
- `--output`：输出目录（默认 ./output）

## 输出

HTML 落地页文件 + 质量检查报告（JSON stdout）。
