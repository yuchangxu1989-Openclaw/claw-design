# Streetwear Bold Design System

## Design System Name
Streetwear Bold Design System

## Style Personality
- Loud, high-contrast, and unapologetic.
- Near-black canvas, acid-green energy, oversized condensed headlines.
- Reads as a drop poster — attitude first.

## Applicable Scenarios
- Work that should grab attention and feel like a brand drop.
- Younger, culture-forward audiences who reward boldness over restraint.
- Use when the requested tone is edgy, hype, or statement-making.

## Brand Colors / Palette
- Canvas: #0a0a0a
- Surface: #161616
- Primary: #d4ff3f
- Accent: #ff3d81
- Text: #f5f5f5
- Muted text: #9a9a9a
- Border: #2a2a2a

## Font Stack
- Heading: "Archivo Expanded", "Anton", Impact, ui-sans-serif, sans-serif
- Body: "Archivo", Inter, ui-sans-serif, system-ui, sans-serif
- Mono: "Space Mono", ui-monospace, Menlo, monospace

## Type Scale
- Caption: 12px / 16px
- Body: 16px / 24px
- Section title: 30px / 36px
- Page title: 64px / 64px

## Spacing System
- Base unit: 8px
- Compact gap: 8px
- Standard gap: 16px
- Section gap: 48px
- Page margin: 40px
- Radius small: 0px
- Radius medium: 0px

## Component Library Path or Component Description
- Components are expressed as HTML/CSS classes generated by Claw Design.
- Use .cd-card for grouped content, .cd-button for primary actions, .cd-chip for status labels, and .cd-panel for dense information regions.
- Cards are hard-edged blocks on near-black surfaces; the acid-green primary and hot-pink accent carry all the energy.
- Prefer CSS variables derived from this file over literal visual values.

## Layout Constraints
- Use an 8px spacing rhythm but allow oversized headline blocks to break the grid intentionally.
- Left align body; headlines may be set large and tight.
- Keep card radius at 0px for a hard, poster-like edge.
- Keep page sections visually distinct without nesting cards inside cards.
- Ensure responsive layouts keep text inside its parent container.

## Reference Examples
- A near-black drop page with a giant acid-green headline, hot-pink tags, and square-edged blocks.
- A product hype layout with oversized type, high contrast, and zero rounded corners.

## Forbidden Rules
- Do not introduce unrelated color families outside the palette.
- Do not use viewport-scaled font sizes.
- Do not round the corners or soften the contrast; the attitude depends on hard edges.
- Do not use pastel or corporate-blue palettes.
- Do not hardcode colors, fonts, spacing, or radius values that conflict with this file.
