# Editorial Mono Design System

## Design System Name
Editorial Mono Design System

## Style Personality
- Print-newsroom discipline with a monochrome backbone.
- Black ink on warm white, a single red flag of emphasis, strong typographic hierarchy.
- Reads as considered, literary, and editorial.

## Applicable Scenarios
- Work that should read like a well-set magazine or long-form feature.
- Audiences who value typography, structure, and a strong point of view.
- Use when the requested tone is editorial, journalistic, or essayistic.

## Brand Colors / Palette
- Canvas: #fbfaf7
- Surface: #ffffff
- Primary: #111111
- Accent: #d1342f
- Text: #161616
- Muted text: #6f6f6f
- Border: #e2e0da

## Font Stack
- Heading: "Playfair Display", Georgia, "Times New Roman", serif
- Body: "Newsreader", Georgia, ui-serif, serif
- Mono: "IBM Plex Mono", ui-monospace, Menlo, monospace

## Type Scale
- Caption: 13px / 18px
- Body: 18px / 30px
- Section title: 28px / 36px
- Page title: 52px / 58px

## Spacing System
- Base unit: 8px
- Compact gap: 8px
- Standard gap: 16px
- Section gap: 40px
- Page margin: 72px
- Radius small: 0px
- Radius medium: 2px

## Component Library Path or Component Description
- Components are expressed as HTML/CSS classes generated by Claw Design.
- Use .cd-card for grouped content, .cd-button for primary actions, .cd-chip for status labels, and .cd-panel for dense information regions.
- Cards are mostly borderless columns separated by rules; the red accent flags pull-quotes and a single call to action.
- Prefer CSS variables derived from this file over literal visual values.

## Layout Constraints
- Use an 8px spacing rhythm with column-style measure and wide outer margins.
- Keep body text left aligned; headlines may sit large with editorial leading.
- Keep card radius near 0px so structure comes from rules and type, not boxes.
- Keep page sections visually distinct without nesting cards inside cards.
- Ensure responsive layouts keep text inside its parent container.

## Reference Examples
- A warm-white feature page with a large serif headline, a single red pull-quote, and 72px margins.
- A long-form article layout driven by typographic hierarchy and hairline rules rather than boxes.

## Forbidden Rules
- Do not introduce unrelated color families outside the palette.
- Do not use viewport-scaled font sizes.
- Do not rely on heavy boxes or large rounded corners for structure.
- Do not use more than one red accent moment per view.
- Do not hardcode colors, fonts, spacing, or radius values that conflict with this file.
