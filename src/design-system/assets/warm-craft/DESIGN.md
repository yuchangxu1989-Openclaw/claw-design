# Warm Craft Design System

## Design System Name
Warm Craft Design System

## Style Personality
- Handmade warmth, tactile and unhurried.
- Soft paper tones, rounded edges, a friendly serif voice.
- Reads as personal, inviting, and made by a human hand.

## Applicable Scenarios
- Work that should feel warm, approachable, and crafted.
- Audiences who respond to personality and storytelling over corporate polish.
- Use when the requested tone is friendly, homemade, or community-led.

## Brand Colors / Palette
- Canvas: #fbf6ee
- Surface: #fffaf2
- Primary: #c2410c
- Accent: #e8a23d
- Text: #3a2e26
- Muted text: #8a7563
- Border: #e7d8c5

## Font Stack
- Heading: Fraunces, Georgia, "Iowan Old Style", serif
- Body: "Source Serif 4", Georgia, ui-serif, serif
- Mono: "IBM Plex Mono", ui-monospace, Menlo, monospace

## Type Scale
- Caption: 13px / 18px
- Body: 17px / 28px
- Section title: 26px / 34px
- Page title: 42px / 50px

## Spacing System
- Base unit: 8px
- Compact gap: 8px
- Standard gap: 20px
- Section gap: 36px
- Page margin: 48px
- Radius small: 10px
- Radius medium: 18px

## Component Library Path or Component Description
- Components are expressed as HTML/CSS classes generated by Claw Design.
- Use .cd-card for grouped content, .cd-button for primary actions, .cd-chip for status labels, and .cd-panel for dense information regions.
- Cards sit on warm paper surfaces with soft 18px corners and a gentle warm border.
- Prefer CSS variables derived from this file over literal visual values.

## Layout Constraints
- Use an 8px spacing rhythm but let sections breathe with warm margins.
- Keep text left aligned; allow a centered title where the artifact invites it.
- Use rounded 18px card radius to keep the handmade feel.
- Keep page sections visually distinct without nesting cards inside cards.
- Ensure responsive layouts keep text inside its parent container.

## Reference Examples
- A cream page with terracotta headings, an amber accent, soft 18px cards, and a serif voice.
- A story page with warm surfaces, generous line height, and a single hand-drawn-feeling highlight.

## Forbidden Rules
- Do not introduce unrelated color families outside the palette.
- Do not use viewport-scaled font sizes.
- Do not use cold neon accents or hard-edged corporate cards.
- Do not flatten the corners; the warmth depends on rounded edges.
- Do not hardcode colors, fonts, spacing, or radius values that conflict with this file.
