# Tech Minimal Design System

## Design System Name
Tech Minimal Design System

## Style Personality
- Restraint over decoration. Precision over flourish.
- Confident negative space, hairline structure, one decisive accent.
- Reads as engineered, calm, and exact.

## Applicable Scenarios
- Work that should feel precise, modern, and uncluttered.
- Audiences that reward clarity and dislike visual noise.
- Use when the requested tone is understated, technical, or product-led.

## Brand Colors / Palette
- Canvas: #ffffff
- Surface: #f6f7f9
- Primary: #0b0b0f
- Accent: #3b82f6
- Text: #0a0a0a
- Muted text: #6b7280
- Border: #e5e7eb

## Font Stack
- Heading: Space Grotesk, Inter, ui-sans-serif, system-ui, sans-serif
- Body: Inter, ui-sans-serif, system-ui, sans-serif
- Mono: ui-monospace, "JetBrains Mono", SFMono-Regular, Menlo, monospace

## Type Scale
- Caption: 12px / 16px
- Body: 16px / 26px
- Section title: 22px / 30px
- Page title: 44px / 52px

## Spacing System
- Base unit: 8px
- Compact gap: 8px
- Standard gap: 16px
- Section gap: 40px
- Page margin: 64px
- Radius small: 2px
- Radius medium: 4px

## Component Library Path or Component Description
- Components are expressed as HTML/CSS classes generated by Claw Design.
- Use .cd-card for grouped content, .cd-button for primary actions, .cd-chip for status labels, and .cd-panel for dense information regions.
- Cards are flat with a single 1px border and no drop shadow; the accent appears at most once per view.
- Prefer CSS variables derived from this file over literal visual values.

## Layout Constraints
- Use an 8px spacing rhythm with generous whitespace between sections.
- Keep text left aligned; reserve centered treatment for a single page title.
- Limit cards to 4px border radius for a sharp, engineered edge.
- Keep page sections visually distinct without nesting cards inside cards.
- Ensure responsive layouts keep text inside its parent container.

## Reference Examples
- A near-white page with one electric-blue accent, hairline borders, 64px margins, and 4px radius cards.
- A product overview with wide whitespace, mono labels for data, and a single highlighted metric.

## Forbidden Rules
- Do not introduce unrelated color families outside the palette.
- Do not use viewport-scaled font sizes.
- Do not use decorative gradient blobs or more than one accent color per view.
- Do not soften the edges with large radii; this system is deliberately sharp.
- Do not hardcode colors, fonts, spacing, or radius values that conflict with this file.
