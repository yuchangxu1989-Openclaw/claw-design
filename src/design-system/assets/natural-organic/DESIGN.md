# Natural Organic Design System

## Design System Name
Natural Organic Design System

## Style Personality
- Calm, grounded, and botanical.
- Sage and clay tones, soft edges, plenty of air.
- Reads as sustainable, wholesome, and quietly premium.

## Applicable Scenarios
- Work that should feel natural, eco-minded, and serene.
- Audiences drawn to wellness, sustainability, and slow living.
- Use when the requested tone is calm, earthy, or restorative.

## Brand Colors / Palette
- Canvas: #f4f1e9
- Surface: #fbfaf4
- Primary: #4f6f52
- Accent: #b07d52
- Text: #2f352c
- Muted text: #77806d
- Border: #ddd9c7

## Font Stack
- Heading: "Spectral", "Cormorant Garamond", Georgia, serif
- Body: "Mukta", Inter, ui-sans-serif, system-ui, sans-serif
- Mono: "IBM Plex Mono", ui-monospace, Menlo, monospace

## Type Scale
- Caption: 13px / 18px
- Body: 17px / 28px
- Section title: 25px / 34px
- Page title: 40px / 50px

## Spacing System
- Base unit: 8px
- Compact gap: 8px
- Standard gap: 16px
- Section gap: 40px
- Page margin: 56px
- Radius small: 8px
- Radius medium: 14px

## Component Library Path or Component Description
- Components are expressed as HTML/CSS classes generated by Claw Design.
- Use .cd-card for grouped content, .cd-button for primary actions, .cd-chip for status labels, and .cd-panel for dense information regions.
- Cards rest on warm off-white surfaces with soft 14px corners and a muted natural border.
- Prefer CSS variables derived from this file over literal visual values.

## Layout Constraints
- Use an 8px spacing rhythm with airy, unhurried section spacing.
- Keep text left aligned; allow a centered title where the artifact invites calm focus.
- Use 14px card radius for a soft, organic edge.
- Keep page sections visually distinct without nesting cards inside cards.
- Ensure responsive layouts keep text inside its parent container.

## Reference Examples
- An oat-colored page with sage headings, a clay accent, soft 14px cards, and a serif voice.
- A wellness layout with breathing room, muted earth tones, and a single grounded highlight.

## Forbidden Rules
- Do not introduce unrelated color families outside the palette.
- Do not use viewport-scaled font sizes.
- Do not use saturated neon or high-gloss corporate accents.
- Do not flatten the corners into a hard technical edge.
- Do not hardcode colors, fonts, spacing, or radius values that conflict with this file.
