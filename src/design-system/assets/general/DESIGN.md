# General Purpose Design System

## Design System Name
General Purpose Design System

## Applicable Scenarios
- General product pages, dashboards, reports, diagrams, posters, and presentation artifacts.
- Use when the user has not supplied a more specific `DESIGN.md` asset.

## Brand Colors / Palette
- Canvas: #ffffff
- Surface: #f8fafc
- Primary: #2563eb
- Accent: #14b8a6
- Text: #0f172a
- Muted text: #475569
- Border: #cbd5e1

## Font Stack
- Heading: Inter, ui-sans-serif, system-ui, sans-serif
- Body: Inter, ui-sans-serif, system-ui, sans-serif
- Mono: ui-monospace, SFMono-Regular, Menlo, monospace

## Type Scale
- Caption: 12px / 16px
- Body: 16px / 24px
- Section title: 24px / 32px
- Page title: 40px / 48px

## Spacing System
- Base unit: 8px
- Compact gap: 8px
- Standard gap: 16px
- Section gap: 32px
- Page margin: 48px
- Radius small: 4px
- Radius medium: 8px

## Component Library Path or Component Description
- Components are expressed as HTML/CSS classes generated by Claw Design.
- Use `.cd-card` for grouped content, `.cd-button` for primary actions, `.cd-chip` for status labels, and `.cd-panel` for dense information regions.
- Prefer CSS variables derived from this file over literal visual values.

## Layout Constraints
- Use an 8px spacing rhythm.
- Keep text blocks left aligned unless the artifact format requires centered title treatment.
- Limit cards to 8px border radius.
- Keep page sections visually distinct without nesting cards inside cards.
- Ensure responsive layouts keep text inside its parent container.

## Reference Examples
- A clean report page with white canvas, slate text, blue primary actions, teal highlights, 48px page margin, and compact 8px radius cards.
- A dashboard with dense panels, clear hierarchy, neutral surfaces, and restrained accents.

## Forbidden Rules
- Do not introduce unrelated color families outside the palette.
- Do not use viewport-scaled font sizes.
- Do not use decorative gradient blobs or one-note color themes.
- Do not use rounded text pills when a standard icon or simple label is clearer.
- Do not hardcode colors, fonts, spacing, or radius values that conflict with this file.
