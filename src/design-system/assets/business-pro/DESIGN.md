# Business Pro Design System

## Design System Name
Business Pro Design System

## Style Personality
- Composed, credible, and quietly authoritative.
- Deep navy structure, a single trustworthy accent, disciplined alignment.
- Reads as enterprise-grade and dependable.

## Applicable Scenarios
- Work that must read as professional, trustworthy, and decision-ready.
- Senior or external audiences who expect polish and restraint.
- Use when the requested tone is formal, corporate, or executive.

## Brand Colors / Palette
- Canvas: #ffffff
- Surface: #f1f5f9
- Primary: #1e3a5f
- Accent: #0e7490
- Text: #111827
- Muted text: #51607a
- Border: #d4dce6

## Font Stack
- Heading: "Libre Franklin", Inter, ui-sans-serif, system-ui, sans-serif
- Body: Inter, ui-sans-serif, system-ui, sans-serif
- Mono: ui-monospace, "IBM Plex Mono", SFMono-Regular, Menlo, monospace

## Type Scale
- Caption: 12px / 16px
- Body: 16px / 24px
- Section title: 24px / 32px
- Page title: 38px / 46px

## Spacing System
- Base unit: 8px
- Compact gap: 8px
- Standard gap: 16px
- Section gap: 32px
- Page margin: 56px
- Radius small: 4px
- Radius medium: 6px

## Component Library Path or Component Description
- Components are expressed as HTML/CSS classes generated by Claw Design.
- Use .cd-card for grouped content, .cd-button for primary actions, .cd-chip for status labels, and .cd-panel for dense information regions.
- Cards use a crisp border and 6px corners; the navy primary anchors headers and the teal accent marks key figures.
- Prefer CSS variables derived from this file over literal visual values.

## Layout Constraints
- Use an 8px spacing rhythm with even, predictable section spacing.
- Keep text left aligned; reserve centered treatment for title slides or covers.
- Limit cards to 6px border radius for a buttoned-up edge.
- Keep page sections visually distinct without nesting cards inside cards.
- Ensure responsive layouts keep text inside its parent container.

## Reference Examples
- A white report with a navy header band, teal key metrics, 56px margins, and tidy 6px cards.
- An executive summary with a clear grid, restrained accents, and confident hierarchy.

## Forbidden Rules
- Do not introduce unrelated color families outside the palette.
- Do not use viewport-scaled font sizes.
- Do not use playful or neon colors, or large rounded corners.
- Do not use decorative gradient blobs or one-note color themes.
- Do not hardcode colors, fonts, spacing, or radius values that conflict with this file.
